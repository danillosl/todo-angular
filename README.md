### Instalação

Este projeto requer node.js, gulp e bower instalados.

Após clonar entre na pasta do projeto e execute os seguintes comandos:

```sh
$ npm install
$ bower install
$ gulp serve
```
Após executar estes comandos o projeto estara disponivel em [http://localhost:3000]
